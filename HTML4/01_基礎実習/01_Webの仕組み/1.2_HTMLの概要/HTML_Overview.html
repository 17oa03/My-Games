1.2_HTMLの概要

▼HTML(HyperText Markup Language)とは、Webページ用の文書を記述する為に開発された言語のことである。コンピューターの環境に依存しない統一した規格を設け、インターネット上で情報を共有しようという目的から考え出された。

▼大きな特色のひとつは、ハイパーリンク機能を持ち、文書のある部分から他の文書へと次々に情報をたどっていくことができるテキストベースの文書である。

▼また、テキストで書かれた文書をコンピューターに理解させる為には、コンピューターにその文書の構造を知らせなければならない。そこで、文書がどのような構成で成り立っているかを、専用のマークを使って「しるし付け」し、文書のもつ情報をコンピューターが読めるようにしたのである。

▼この HTML は、W3C(World Wide Web Consortium)という非営利団体が、仕様の協議決定を行っている。Web 技術の標準化とその推進を目的として活動している W3C は、1994年10月、MIT(マサチューセッツ工科大学)に設立された。現在では ERCIM(欧州情報処理数学研究コンソーシアム)と慶應義塾大学SFC を加えた三者がホスト組織として共同運営している。
