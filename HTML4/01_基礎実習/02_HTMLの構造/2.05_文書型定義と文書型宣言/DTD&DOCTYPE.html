2.5_文書型定義と文書型宣言

▼HTML では使用される要素や属性の特色や用法などをバージョン毎に文書型定義(Document Type Definition : DTD)として詳細に定義している。

▼実際に HTML 文書を作成する場合、どのバージョンに基づいた文書であるのかを宣言する必要がある。これを文書型宣言(DOCTYPE 宣言)といい<!DOCTYPE～>の書式で記述する。

▼HTML4.01 では、構造と見栄えの分離を目指しつつも、当面は以前のバージョンとの互換性も考慮しようという姿勢から、3種類の文書型定義が用意されている。 

1)HTML4.01 Strict DTD : <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

2)HTML4.01 Transitional DTD : <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

3)HTML4.01 Frameset DTD : <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">
