### ゲーム説明

左右キーで自機が左右に移動します、  
また、スペースキー( space key )で攻撃できます。  
攻撃した際に発射される魔法を敵機に当て、  
敵機を撃墜してスコア( SCORE )を伸ばすゲームです。
